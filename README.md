# ohmyzsh_custom

Modify **.zshrc**

```shell

# Set name of the theme to load
ZSH_THEME="virink"

# custom folder than $ZSH/custom
ZSH_CUSTOM=/Users/yourname/path/ohmyzsh_custom

# Standard plugins
plugins=(
    osx
    git
    github
    dotenv
    pip
    yarn
    autojump
    sublime
    docker
    docker-compose
    gem
    mvn
    # Custom plugins
    zsh-syntax-highlighting
    zsh-autosuggestions
    virink
)
```
